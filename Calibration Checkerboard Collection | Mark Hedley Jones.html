Calibration Checkerboard Collection

2018-04-15

Checkerboards can be used with camera calibration tools such as the camera_calibration package for ROS. You can freely download a wide range of checkerboards, with no watermarks or advertising, in PDF format by clicking any of the links below. Additionally, a javascript based checkerboard generator has been added that allows you to generate custom sizes.

TIP: Make sure to disable any page scaling or automatic page fitting features when printing otherwise the checker dimensions will be incorrect.

Generate Your Own Checkerboards

This script generates Scalable Vector Graphic (SVG) files, which can be edited using free software such as Inkscape.

Saving as PDF:
  • Click the generated thumbnail to download as SVG
  • Open the downloaded checkerboard in your browser
  • Print the document (File -> Print)
  • Select the Save to PDF
  • Set scale to 100%, (uncheck Fit-to-page or similar options) and disable any headers & footers
  • Choose desternation to save the PDF
Feel free to get in touch if you have any questions or issues.

WARNING: Your browser will become unresponsive if the dimensions become too extreme
